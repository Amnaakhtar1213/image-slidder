##  3D Image Slider - Korea Theme 🎠

A visually engaging **3D rotating image slider** built with **HTML & CSS**.  
This project showcases Korean culture through a carousel of images, animated in 3D space with smooth rotation and gradient styling.

---

##  Features ✨
- **3D Carousel Effect**: Images rotate around the Y-axis with perspective depth.
- **Auto Rotation**: Continuous animation using CSS `@keyframes`.
- **Gradient Background**: Inspired by the Korean flag colors (red → white → blue).
- **Custom Fonts**: Uses Google Fonts (`Roboto`, `Gravitas One`, `Dancing Script`) for stylish text.
- **Responsive Layout**: Centered banner with flexible positioning.
